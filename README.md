# 📘 Learning C++

A repository documenting my journey of learning **C++ programming**.
It contains small programs written while exploring core C++ concepts and experimenting with different language features.

---

## 🚀 Topics Covered

This repository includes practice programs for:

* Basic C++ syntax
* Functions and parameters
* Function overloading
* Recursion
* Pointers and references
* Structures
* Arrays and matrices
* Object-Oriented Programming

## ⚙️ Compile and Run

Using **g++**:

```bash
g++ filename.cpp -o program
./program
```

Example:

```bash
g++ factorial.cpp -o factorial
./factorial
```
